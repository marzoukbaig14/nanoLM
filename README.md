# nanoLM

A character-level language model built from scratch in PyTorch, 
starting from a simple bigram model and building toward a transformer.

Implements the full progression:
- Bigram language model (counting-based and neural network equivalent)
- MLP with character embeddings
- Transformer (in progress)

Built to understand how modern language models work at the fundamental level.

## What's here

- `bigram.py` — bigram model: count-based and single-layer NN equivalent
- `demo.ipynb` — full walkthrough notebook
